# Excalivault

Personal vault for storing Excalidraws - A Chrome extension built with WXT, Svelte 5, Tailwind 4, and DaisyUI.

## Quick Start

1. Install from the [Chrome Web Store](#)
2. Open the side panel
3. Start saving your Excalidraws!

## Features

- 📁 **Folder Organization** - Organize your drawings in folders
- 💾 **Local Storage** - All data stored locally in your browser
- 🔒 **Privacy First** - No data sent to any servers
- 🎨 **Thumbnails** - Preview your drawings with 80x80 thumbnails
- ⚡ **Fast & Lightweight** - Built with performance in mind
