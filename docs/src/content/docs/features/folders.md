# Folder Organization

Excalivault lets you organize your drawings into folders for easy access.

## Creating Folders

1. Open the Excalivault side panel
2. Click the "New Folder" button
3. Give your folder a name
4. Choose a parent folder if you want nested organization

## Folder Colors

Folders can be assigned colors for visual distinction:

- Amber
- Mint
- Rose
- Sky
- Leaf
- Coral
- Indigo
